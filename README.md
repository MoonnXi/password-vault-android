# 密码保险箱（Password Vault）

一个安全的密码管理Android应用，支持随机密码生成、密码封存和历史记录查看功能。

## 功能特点

- **随机密码生成**：支持自定义密码长度、包含字符类型（小写字母、大写字母、数字、特殊字符）
- **密码封存**：将生成的密码封存，并设置解锁时长（1分钟、15天、25天、30天）
- **倒计时显示**：封存后显示中文格式的倒计时（如"1天2时3分4秒"）
- **历史密码记录**：查看历史生成的密码，支持滚动查看所有记录
- **离线支持**：当网络不可用时，仍可使用离线模式生成密码

## 安装步骤

### 方法一：直接安装APK
1. 下载项目构建的APK文件
2. 在Android设备上安装（可能需要开启"未知来源"安装权限）

### 方法二：从源码构建
1. **克隆仓库**
   ```bash
   git clone https://github.com/你的用户名/password-vault-android.git
   cd password-vault-android
   ```

2. **使用Android Studio打开项目**
   - 启动Android Studio
   - 选择"Open an existing project"
   - 导航到克隆的项目目录并打开

3. **构建并运行**
   - 连接Android设备或启动模拟器
   - 点击"Run"按钮构建并安装应用

## 使用方法

1. **首次启动**：进入密码配置页面，设置密码长度和包含字符类型

2. **生成密码**：
   - 在主页面点击"生成随机密码"按钮
   - 系统会根据你的配置生成一个随机密码

3. **复制或封存密码**：
   - 生成密码后，可以选择"复制密码"直接复制到剪贴板
   - 或选择"封存密码"，设置解锁时长后封存

4. **查看历史密码**：
   - 在密码配置页面或倒计时页面点击"查看历史密码"按钮
   - 查看所有历史生成的密码记录

5. **解锁封存的密码**：
   - 等待倒计时结束
   - 倒计时结束后，系统会自动跳转到密码提取页面
   - 在提取页面复制密码

## 技术栈

- **开发语言**：Java
- **开发工具**：Android Studio
- **存储**：SharedPreferences（本地存储）
- **网络**：自定义API客户端（用于在线密码生成）

## 注意事项

- **安全性**：本应用仅在本地存储密码，不会上传到任何服务器
- **离线模式**：当网络不可用时，会自动切换到离线模式生成密码
- **历史记录**：历史密码记录存储在本地，建议定期清理不需要的记录
